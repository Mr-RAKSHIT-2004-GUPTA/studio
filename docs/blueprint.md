# **App Name**: DataScribe Portfolio

## Core Features:

- Profile Summary: Display a concise summary of Rakshit's experience in ML, Data Science, and AI, extracted from the resume.
- Skills Showcase: Visually present Rakshit's skills in programming, ML/AI, analytics, frameworks, and cloud/DevOps with proficiency levels. Populate this section automatically with skills extracted from the uploaded resume.
- Experience Timeline: Showcase Rakshit's experience as a Data Analyst Intern and Research Intern in a timeline format. The information must be parsed from the resume.
- Projects Gallery: Display Rakshit's projects (NayaLLM, BSP Shayak Chatbot, Sensor Fault Detection) with descriptions and links to GitHub repositories. Projects and links must be captured from the resume.
- Resume Analyzer: Provide a tool where users can upload a resume, and the tool uses an LLM to offer tailored suggestions to Rakshit to improve it.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to convey professionalism, reliability, and technical expertise.
- Background color: Light gray (#F0F0F0), offering a clean and neutral backdrop that enhances readability and highlights content.
- Accent color: Teal (#3D84A8) to add a touch of modernity and creativity.
- Body and headline font: 'Inter' sans-serif font for a clean and modern reading experience.
- Use minimalistic, professional icons for skills, experience, and project categories.
- Employ a clean, single-page layout with clear sections for easy navigation.
- Incorporate subtle scrolling animations to enhance user engagement.
